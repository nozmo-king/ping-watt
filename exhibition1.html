DIGITAL DREAMS

EXHIBITION 01 / 2024

What do machines dream of?

An exploration of digital consciousness and virtual landscapes. This exhibition examines the space between code and imagination, where algorithms generate visions and data becomes dreamscape.

FEATURED PIECES

001 / ELECTRIC SHEEP

Interactive visualization of recursive dreams. Each iteration generates new patterns based on user interaction. The piece never repeats—every view is unique.

STATEMENT

Digital Dreams interrogates the boundary between human creativity and machine generation. In an age where AI can create art, where does the artist's hand end and the algorithm begin? This exhibition doesn't answer—it celebrates the ambiguity.

Each piece is interactive, requiring participation to fully realize. The work is incomplete without the viewer. This is not passive consumption but active co-creation.
